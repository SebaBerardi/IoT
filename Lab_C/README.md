### Parte 1

- Pararse en el directorio `Parte_1` y ejecutar el siguiente comando:
```bash
.\build.cmd compile
```

- Para ejecutar lo compilado:
```bash
.\build.cmd run
```

Esto desplegará el resultado de ejecutar algunas funciones de prueba.
